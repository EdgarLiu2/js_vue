# Vue.js 101

Welcome to your first Vue.js tutorial in CodeMix!

# Installing Vue on Mac


```bash
brew update
brew install node

node -v
npm -v

npm install -g @vue/cli
npm install -g webpack

vue ui
npm run serve

npm install
```
